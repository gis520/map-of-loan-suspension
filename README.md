# map-of-loan-suspension

数据来源：https://github.com/WeNeedHome/SummaryOfLoanSuspension

技术栈：`Vue.js + 高德 Web JS`

在线 demo 预览：http://xiaomiquan.gitee.io/map-of-loan-suspension

![](./demo1.png)

## Usage

node 版本建议 `v14`，避免 node-sass 乱报错

- `npm install`
- `npm run start`
